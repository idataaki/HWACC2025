# Quizzes

## Overview

This folder contains 5 quizzes based on research papers assigned for student presentations.  
Each quiz has 4 questions focused on the **core ideas and key concepts** of the papers listed in [Selected Articles](../Papers/SelectedArticles.md).

## Why These Quizzes?

Quizzes are taken **before presentations** to ensure papers are read in advance. This can build a shared baseline for discussion in the class and encourage active participation during class.
