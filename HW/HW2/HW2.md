## Overview

This assignment focuses on implementing a **Multilayer Perceptron (MLP)** neural network for classification tasks. Students will implement the feedforward and backpropagation algorithms from scratch and then create a hardware implementation for inference.



## Dataset

**NIST Handwritten Digits Dataset**
- 1797 samples of 8x8 grayscale images
- 64 features (one per pixel), digits 0-9
- Preprocessed data provided in `optdigits.tes` file
- Each row contains 64 pixel values followed by the label



## Network Architecture

The MLP network consists of:
- **Input Layer:** 64 inputs (8x8 pixel values)
- **Hidden Layer:** 256 neurons with ReLU activation function
- **Output Layer:** 10 neurons (one per digit class) with Sigmoid activation

## Part 2: Hardware Implementation

### Objective
Implement the trained MLP network in hardware for inference only (no training required).

### Requirements

1. **Hardware Description Language:** Use Verilog/VHDL
2. **Design Tool:** Xilinx Vivado
3. **Implementation Scope:**
   - Implement only feedforward (inference) phase
   - Load trained weights from Part 1

### Design Specifications

- Successfully complete synthesis and implementation in Vivado
- Report schematic diagrams generated by Vivado
- Report power consumption and other Vivado-generated parameters
- Read test data from dataset files in HDL code

### Evaluation

- Calculate accuracy on test dataset
- Compare hardware implementation accuracy with software version
